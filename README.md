# chat-tool
chat-tool created with Nodejs, MongoDB, Vuejs and socket.io <br>
Auto documentation is created with `swagger`

- frontend: port `8080`
- backend: port `3000`

## How to start 
- Go to each directory for frontend and backend separately 
- frontend: 
```
npm run serve
```
- backend:
```
node index 
```
